Picking Yourself Up by Your Bootstraps

Luckily for us, there are CSS libraries that will automatically format our website for us. The one we will learn today is 'Bootstraps.' It is very popular, and even the submission site is styled with Bootstraps. How do we use Bootstraps? We specify in the class of the object was type of formating we want. This allows the Bootstraps style sheet to load and simplifies our work by a ton. For a full list of Bootstrap compnents, visit the Components page. Basically, to format our webpage, we change the classes of our webpage to match the ones provided by Bootstraps, and then we edit the CSS to our own liking. Let's run through a few examples.

Bootstraps was also made to be mobile friendly, so components automatically resize depending on the size of the screen. We will not be exploring more into mobile compatibility, but you may wish to do more on your own.
