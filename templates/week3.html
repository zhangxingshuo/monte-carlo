Pretty Pretty, Shiny Shiny

So far, our webpages have been very plain and boring. Or, in the case of the Superman example, just plain ugly. The way web developers make better web pages is by using Cascading Style Sheets (CSS). CSS gives us many advantages over only using HTML. Today, we will use CSS to beautify our website.

Advantages of CSS

We mentioned that CSS is like the butter to HTML's toast, but why is that?

  • Editing Power: CSS gives one the power to edit every attribute on their web page in so many ways, there is no need to learn them all. Many ways you could want to change a page can be done using a few lines of CSS.

  • Maintenance: Changes can resonate throughout your entire website, including on different pages, by only changing one file. If you want all the paragraphs on your website to change from black to blue, this can be achieved relatively easily using external style sheets.

  • The W3C Group says so: The W3C maintains CSS and recommends how it should change. Many HTML tags that were used to modify web pages have been deprecated, or considered obselete.

The Basics

Let us start with the basics. A lot of CSS does exactly what it says it will. To understand what is happening, simply use your old friend Inspect Element!

Adding Color

Background Color

By setting the background color to "yellow," we change the area behind this text. We can also change the attribute several other ways, with the most popular being hex. The hex representation of yellow in hex is #FFFF00. Hex gives of the ability to change to colors that might not have names.

See? It worked here, too. Hex code is very hard to understand as a human, or at least at first. It's like reading a paper in binary, a big inconvenience. We will now learn the basics about it. Every two numbers denotes how much red, green, and blue is in the whole color. The first two are for the amount of red, the next for green, and the last ones are for blue. The max value for a color is FF (255) and the min value is 00 (obviously 0). From there, you can mix colors like you used to in elementary school, with different amount of each color. Or, you could search hex values of different colors and see which one works the best.

We may also want to change which font family we are displaying. The reasons for doing so can range from simply a prettier font style to readibility. Typically sans-serif fonts are eaier to read on screens, but are not regarded to be as fancy. A serif font has letters that have accents on the tips.

Selectors

A selector is a HTML tag that CSS will modify. For external and embedded style sheets, the form to modify a selector is:

selector {
  attribute: value;
}

We can make every instance of an HTML tag the selector by simply puting placing it in the specified selector place.
