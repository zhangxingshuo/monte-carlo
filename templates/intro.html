Webpage Magic

Behind every website on the Internet, there is an intricate web of code that allows it to run smoothly and efficiently, all while looking pretty! Most of these under-the-hood operations go unnoticed and neglected :( and thus, are rarely understood by the casual user. However, we in CS 5 are not the casual user! We are going to delve into the inner-workings of websites using tools already built into your browser! With these tools, you'll be able to do things like change the text, colors, and images on any webpage!

Every browser comes with a handy tool called the Web Inspector which can help you, the developer, get a better grasp of the code behind any website. This week, your task is to delve into the web inspector built into your browser and change a page! The examples in this lesson are done using Google Chrome, but there are similar looking web inspectors in browsers all across the board. To follow along, you'll probably want to know where to find Inspect Element, and that is a quick process outlined here. Or, you can usually use F12 to access it.

Now, the first thing you will notice when opening web inspector is the different portions of the page that light up when you move your mouse over the code. This is an extremely handy tool because it shows you what part of the web page the code is referring to. Let's take this spammy website. for example:

As you might be able to infer, the large elements of the page are divided up into sections marked by the <div> tag. Your intuition might tell you that the best place to edit the text on the page is in the class "body" Open up the "body" tab and explore all the "<h(number)> </h(number)>" "<p> </p>" markers (you'll learn what these markers mean next week). In it, you'll find some text that is also found on the web page itself! A simple double click on the text will allow you to change it. The text change will then be reflected on the web page in real time!
